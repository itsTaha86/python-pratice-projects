# to-do-web

Simple to do website
To-Do can read your notes from excel or save your notes as JSON

Excel sample located on (to-do-web/samples/notes_sample.xlsx)
Your excel notes should be same as sample: (Title | content | done)

JSON data always located in data, you can download it from application
Also you can replace your JSON data in (to-do-web/data/notes.json)

To use this application install requirements:
```bash
pip install -r requirements.txt
